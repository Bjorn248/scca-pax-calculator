Welcome to the SCCA PAX Calculator!

This tool is designed to help you figure what kind of raw time you need to have a better PAX time than a competitor in a different class, given their raw time.


DISCLAIMER: This is not an official SCCA website. This is a third-party website. This website is automatically generated once a month using the PAX Index as the source of truth.
